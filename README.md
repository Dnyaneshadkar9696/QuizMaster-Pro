# QuizMaster-Pro

QuizMaster-Pro is an advanced quiz management application designed for creating, managing, and taking quizzes with ease. With a user-friendly interface and robust features, it's ideal for educators, students, and anyone interested in learning or testing knowledge interactively.

## Features

- Create unlimited quizzes with multiple question types (MCQs, True/False, Short Answer, etc.)
- Organize questions into customizable categories
- Timer features and scoring system
- User authentication and performance tracking
- Responsive and intuitive interface
- Export and import quiz data

## Getting Started

### Prerequisites

- Node.js (v14 or higher recommended)
- npm or yarn package manager

### Installation

1. Clone the repository:
    ```bash
    git clone https://github.com/Dnyaneshadkar9696/QuizMaster-Pro.git
    cd QuizMaster-Pro
    ```

2. Install dependencies:
    ```bash
    npm install
    # or
    yarn install
    ```

3. Start the development server:
    ```bash
    npm start
    # or
    yarn start
    ```

4. Visit `http://localhost:3000` in your browser to use the app.

## Usage

- To create a quiz, navigate to the "Create Quiz" section and add questions.
- Manage your quizzes from the dashboard.
- Share quiz links with participants or take quizzes yourself.
